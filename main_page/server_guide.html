Hama Server Guide

Lobby Server

처음 접속하는 서버로, 광장에서 다른 플레이어와 소통하고 거래를 할 수 있습니다. 물물교환, 상점 시스템을 통해 재화를 거래하고 서버의 중심이 되는 공간입니다.

Farm Server

개인 농장을 가꾸고 다양한 농작물을 재배할 수 있는 서버입니다. 수확한 농작물은 서버 내에서 화폐로 교환할 수 있어 경제활동의 중요한 역할을 합니다.

Fish Server

물고기를 낚을 수 있는 서버로, 낚은 물고기를 화폐로 교환해 다양한 경제활동에 참여할 수 있습니다. 낚시를 통해 다양한 물고기를 수집할 수 있습니다.

Wild Server

마인크래프트의 정통 야생 모드 서버로, 플레이어는 생존을 목표로 게임을 진행합니다. 자원을 모아 생존 전략을 세우고, 끝없이 확장되는 세계를 탐험할 수 있습니다.
